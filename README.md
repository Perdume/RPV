# RPV — 숫자 게임 (Numbers Game)

RPV는 **턴제 전투 전략 게임**입니다. 플레이어들은 매 턴 행동을 선택하고, 각자 고유한 능력을 활용하여 마지막까지 살아남는 것을 목표로 합니다.

## 게임 개요

- **장르**: 턴제 전략 / 배틀 로얄
- **플랫폼**: Electron 데스크탑 앱 (Windows)
- **언어**: TypeScript + React

## 주요 특징

- 🎮 **다양한 고유 능력**: 25개 이상의 개성 넘치는 능력, 각자 독특한 전략을 구사
- ⚔️ **턴제 전투**: 공격·방어·회피·패스 중 선택하는 간결한 액션 시스템
- 👻 **심판자 시스템**: 비활동 플레이어에게 패널티를 부과하는 자동 심판 NPC
- 📊 **상태이상**: 균열, 피해 감소, 능력 봉인 등 다채로운 버프/디버프
- 💾 **데이터 저장**: 능력 사용 기록이 게임 세션 간에 유지

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (웹)
npm run dev

# Electron 앱으로 실행
npm start
```
