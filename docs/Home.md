## 🎮 게임 한눈에 보기

RPV는 **여러 플레이어가 매 턴 행동을 선택**하여 싸우는 전략 게임입니다.  
각 플레이어는 게임 시작 전 **고유한 능력 하나**를 부여받으며, 이 능력이 게임의 핵심 전략이 됩니다.

### 기본 흐름

```
[게임 시작]
    ↓
각 플레이어에게 능력 배정
    ↓
[턴 시작]
    ↓
모든 플레이어 행동 선택
(공격 / 방어 / 회피 / 패스 / 능력 사용)
    ↓
동시에 행동 공개 & 처리
    ↓
상태이상, 피해, 회복 적용
    ↓
탈락 여부 확인
    ↓
[다음 턴] 반복 → 최후의 1인이 승리
```

### 핵심 개념 요약

| 용어 | 설명 |
|------|------|
| **HP (체력)** | 0이 되면 탈락 |
| **사용 횟수** | 능력을 쓸 수 있는 횟수. 다 쓰면 능력을 쓸 수 없음 |
| **균열** | 쌓이면 피해를 받는 디버프 |
| **능력 봉인** | 다음 턴 능력을 사용할 수 없는 디버프 |
| **패시브** | 능력을 선택하지 않아도 자동으로 발동되는 효과 |
| **심판자** | 행동하지 않는 플레이어에게 자동으로 패널티를 주는 특수 NPC |
